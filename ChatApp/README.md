## Installation Guide

### Requirements
- [Nodejs] (Used ReactJS)
- [Mongodb] (For the database)


```shell
git clone https://github.com/Dabiidi/ChatApp

cd ChatApp
```
Install the dependencies 
```shell
cd serverSide
yarn
cd ..
cd clientSide
yarn
```
Start the server.

For the frontend.
```shell
cd clientSide
yarn start
```
For the backend.

```shell
cd serverSide
yarn start
```
